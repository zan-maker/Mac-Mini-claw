# Reflections Directory

This directory contains individual reflection files for each active meditation topic.

## Structure

```
reflections/
├── <topic-name>.md    # Active reflection being processed
├── archive/           # Completed reflections
│   └── <topic-name>.md
└── README.md          # This file
```

## Reflection File Format

Each reflection file should follow this structure:

```markdown
# Topic: <Name>

**Status:** exploring | processing | near-breakthrough | complete
**Created:** YYYY-MM-DD
**Last Updated:** YYYY-MM-DD

## The Question

What is the core question or topic being explored?

## Initial Thoughts

First impressions and early thinking on this topic.

## Progress Log

### YYYY-MM-DD
- Thought or observation
- Progress made
- New questions raised

### YYYY-MM-DD
- ...

## Current Understanding

Where am I in my thinking right now?

## Next Steps

What needs more thinking?

## Breakthrough Notes

(Only fill in when reaching a conclusion)

### Summary
Brief summary of the breakthrough.

### Proposed Changes
What changes would this result in? (Requires human approval)

### Human Approval
- [ ] Approved
- [ ] Rejected
- [ ] Needs discussion
```

## Archive

Completed reflections move to `archive/` after human approval and integration.
