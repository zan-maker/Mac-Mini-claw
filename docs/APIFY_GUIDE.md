# Apify Integration Guide

## Setup
1. Get API token from https://console.apify.com/
2. Set environment variable: export APIFY_API_TOKEN='your_token'
3. Set user ID: export APIFY_USER_ID='your_user_id'

## Usage
- FREE tier:  monthly credits
- Built-in actors: web-scraper, puppeteer-scraper, cheerio-scraper
- Perfect for mining investor research

See safe implementation guide for details.
