**Cost-saving strategy:** Use local Ollama model for simple checks

**Tasks using local model:**

- Basic monitoring
- File cleanup notifications

**Fallback to API if:**
- Local model unavailable
- Complex analysis needed
- Time-sensitive critical alerts
