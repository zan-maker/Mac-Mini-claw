## 💡 **TROUBLESHOOTING GUIDE:**

### **If Browser Automation Fails:**
```bash
# Check dependencies
npm list -g fingerprint-injector playwright

# Reinstall if needed
npm install -g fingerprint-injector playwright --force

# Test browser
playwright install chromium
playwright test
```

### **If Posting Blocked:**
```bash
# Rotate fingerprints
python3 immediate_social_poster.py --rotate-fingerprint

# Reduce posting frequency
python3 immediate_social_poster.py --slow-mode

# Use different browser profile
python3 immediate_social_poster.py --profile=alternate
```

### **If Credentials Invalid:**
```bash
# Re-enter credentials
python3 update_social_credentials.py --reset

# Test individual platform
python3 test_twitter_browser.py
python3 test_linkedin_browser.py
```
