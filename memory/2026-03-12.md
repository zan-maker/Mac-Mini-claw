### **🔐 STRIPE API KEY CLARIFICATION:**

**User Question:** "what sort of Stripe API is needed Restricted Key, Publishable Key or Secret Key?"

**✅ ANSWER PROVIDED:** Detailed breakdown of Stripe API keys needed:

**For PlumberPro/BuilderBase ($99-149/month subscriptions):**

**1. 🔑 SECRET KEY (Server-side - MOST IMPORTANT)**
- **Purpose:** Backend operations (charges, subscriptions, webhooks)
- **Location:** Backend server environment variables **ONLY**
- **Security:** **NEVER** expose in frontend code, GitHub, or browser

**2. 🎫 PUBLISHABLE KEY (Client-side)**
- **Purpose:** Frontend tokenization (card collection)
- **Location:** Frontend JavaScript code
- **Security:** Safe to expose publicly (designed for this)

**3. 🔒 RESTRICTED KEY (Optional but Recommended)**
- **Purpose:** Limited backend operations (more secure than Secret Key)
- **Location:** Backend server environment variables
- **Permissions:** Customizable - restrict to specific operations

**Minimum Required:**
1. ✅ PUBLISHABLE KEY (`pk_live_...`) - For frontend
2. ✅ SECRET KEY (`sk_live_...`) - For backend

**Recommended:**
3. ✅ RESTRICTED KEY (`rk_live_...`) - Replace Secret Key for better security
4. ✅ WEBHOOK SECRET (`whsec_...`) - For webhook verification
